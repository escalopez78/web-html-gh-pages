Primer articlulo - ¿Que es HTML?

Todo sobre HTML

HTML, acrónimo en inglés de HyperText Markup Language
(«lenguaje de marcado de hipertexto»), hace referencia
al lenguaje de marcado utilizado en la creación de páginas web.

Segundo articulo - ¿Que es CSS?

Todo sobre CSS

CSS (siglas en inglés de Cascading Style Sheets; en español:
«hojas de estilo en cascada») es un lenguaje informático
especializado en definir y cohesionar la presentación
de un documento escrito en un lenguaje de marcado como
HTML o XML
